✅ 1 – Torre de Hanói
- Tema: Recursão
- Descrição: Implementar o algoritmo clássico da Torre de Hanói usando três hastes (A, B, C).
- Objetivo: Mover todos os discos da haste A para a haste C, usando a haste B como auxiliar, respeitando as regras:
  - Só um disco pode ser movido por vez.
  - Um disco maior nunca pode ficar sobre um menor.
- Desafio principal: Implementar a lógica recursiva corretamente.

✅ 1 – Tower of Hanoi
- Topic: Recursion
- Description: Implement the classic Tower of Hanoi problem using three rods (A, B, C).
- Goal: Move all the disks from rod A to rod C using rod B as auxiliary, following these rules:
  - Only one disk can be moved at a time.
  - A larger disk can never be placed on top of a smaller one.
- Main challenge: Correctly implement the recursive logic.
